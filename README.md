{% hint style="info" %}
There are two methods to return your result.

`result = { "data": YOURVARIABLE }`

Or just:

`return { "data": YOURVARIABLE }`

Both methods will return the result in the JSON\
`{ "data": YOURVARIABLE }`
{% endhint %}
